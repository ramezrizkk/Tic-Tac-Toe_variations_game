Detailed Description

Diamond-shaped Tic-Tac-Toe variant on 7x7 grid.

Provides gem_grid, human_participant, ai_participant, and gem_interaction classes for a game where players need both 3-in-a-row and 4-in-a-row in different orientations to win.
